Build instructions:

    >mkdir photofilters
    >cd photofilters/
    >git clone https://github.com/UNN-VMK-Software/itlab-vision.git
    >mkdir build
    >cd build/
    >cmake ../itlab-vision/photoeffects/
    >make
    >cd ./bin/

Tests execution:

    >./photoeffects_test
